In-depth information about the AWS re/start program.

☁️

AWS re/start

Eligibility: Applicant must be a STEM Graduate / BCA / MCA / BSc IT / B Tech / M Tech / BE.

⏱️ 12 weeks

AWS re/Start is a full-time, skills development program that prepares learners for careers in the cloud. Through real-world, scenario-based learning, hands-on labs, learners gain the technical skills they need for entry-level cloud roles. AWS re/Start also focuses on building professional skills such as adaptive communication, time management, and collaboration. The program’s mission is to build a diverse pipeline of entry-level cloud talent.

Domain Skills:

  • Fundamentals of cloud computing, AWS Cloud skills
  • How to use different programming languages
  • Build skills aligned with AWS certification - Prepare for the AWS Certified Cloud Practitioner Certification
Employability Skills: Effective communication, Problem solving, Teamwork, Time management

Centres offering this course:

  • Tata STRIVE Extension Centre – Nagavara Bengaluru (Model Name: 02-TSEC TS)
  • Tata STRIVE Extension Centre – Dwarka Delhi (Model Name: 02-TSEC TS)
  • Tata STRIVE Mumbai (Model Name: Tata STRIVE Skill Development Centre)
  • Tata STRIVE Extension Centre Gurugram (Model Name: 02-TSEC TS)

Commis Chef

Eligibility:

  • 12th Class/I.T.I (two years after class 10th) OR
  • 12th Class/I.T.I (one year after class 10th with 1 year of experience in the food production sector) OR
  • Certificate-NSQF (level-3 Assistant Chef)
This course provides you with the knowledge and skills required to work as a Commis Chef in the Tourism & Hospitality industry.

⏱️ 11 weeks classroom + 8 weeks OJT

Domain Skills:

  • Set up kitchen for operations
  • Prepare sauces, salads, cold starters and other base items
  • Monitor stocks, assist Commis 1 or Chef-de-Partie
  • Close kitchen at end of day’s operations or shift-end
Employability Skills: Communication; Work Place Readiness; Life Skills; Health, Hygiene and Safety Practices

Centres offering this course (Model Name: 02-TSEC TS):

🏢 About Tata STRIVE Dwarka

Learn more about Tata STRIVE's mission, leadership, and governance

Tata STRIVE is a Skill Development programme housed under the overall governance of Tata Community Initiatives Trust (TCIT), which was established in October 2014 by Tata Sons Private Limited. Today, Tata STRIVE functions under the aegis of the Tata Trusts. TCIT is governed by a Board of Trustees.
